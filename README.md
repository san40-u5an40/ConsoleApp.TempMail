## Назначение
Консольное Windows-приложение для использования сервиса временной почты `mail.gw`. Данный сервис предоставляет открытый [API](https://api.mail.gw/) и подробную [документацию](https://docs.mail.gw/) по его использованию.

## Примеры работы с приложением
Запуск приложения:
```CMD
Z:\>temp_mail :: Если приложение называется "temp_mail.exe" и находится в $PATH, иначе потребуется указание полного пути к приложению
```

После этого запустится дисплей консольного приложения. Для навигации необходимо использовать стрелки ↓ ↑ и клавишу [enter] для выбора пункта меню.
```CMD
/*-------------------------------------------------------------------------------------------*/
/*-------------------------------Добро пожаловать в программу--------------------------------*/
/*--------------------------------------TempMailClient---------------------------------------*/
/*-------------------------------------------------------------------------------------------*/

Меню:
1) Войти в существующий аккаунт ←
2) Создать аккаунт
3) Выйти из приложения
```

Если вы ранее создавали аккаунт, можно указать логин и пароль в аргументах приложения:
```CMD
Z:\>temp_mail test@teihu.com simple_password
```

После успешной аутентификации будет выведено основное окно приложения:
```CMD
/*-------------------------------------------------------------------------------------------*/
/*-------------------------------Добро пожаловать в программу--------------------------------*/
/*--------------------------------------TempMailClient---------------------------------------*/
/*-------------------------------------------------------------------------------------------*/

Меню сообщений test@teihu.com:

1) Выйти из приложения ←
2) Удалить аккаунт
3) Обновить список

Количество сообщений: 1
Это сообщение обязательно увидят!     | Александр Чалышев        | 07.11.2025 19:01     | del
```

Для просмотра сообщения его необходимо выбрать и также нажать [enter]. Для удаления необходимо выбрать пункт `del` справа от сообщения при помощи соответствующей стрелки.

## Компиляция исполняемого файла

### Установка SDK

SDK-пакет необходим для разработки и компиляции .NET/C#-приложений. Официальный [сайт](https://dotnet.microsoft.com/ru-ru/download) для скачивания.

После установки SDK-пакета убедитесь в её успешности при помощи команды:
```
dotnet --info
```

### Создание исполняемого файла
После успешной установки пакета-SDK можно скомпилировать данный проект. Для этого необходимо перейти в его директорию, в подпапку `src`, и запустить следующую команду в консоли:
```
dotnet publish -c Release
```

При успешной компиляции исполняемый файл можно будет забрать по следующему пути: `.\ConsoleApp.TempMail\src\TempMailApp\bin\Release\net9.0\win-x64\publish\`.

Так как программа компилируется в один исполняемый файл, получившийся *.exe-файл можно перемещать в любую директорию и беспрепятственно его запускать.
